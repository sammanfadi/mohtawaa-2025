# قالب Muhtawaa - دليل التثبيت والاستخدام

## نظرة عامة

قالب **Muhtawaa** هو قالب ووردبريس مخصص للمحتوى المعرفي القصير، مصمم خصيصاً لموقع muhtawaa.com. يركز القالب على تقديم تجربة قراءة مريحة وبسيطة للمقالات القصيرة (90 ثانية قراءة).

## الميزات الرئيسية

### ✨ التصميم والواجهة
- تصميم نظيف ومتجاوب مع جميع الأجهزة
- دعم كامل للغة العربية مع اتجاه RTL
- خطوط عربية محسنة (Tajawal, Cairo, Almarai)
- نظام ألوان قابل للتخصيص
- الوضع الليلي مع إمكانية التحكم

### 📱 تجربة المستخدم
- تخطيط بطاقات للمقالات مع معاينة
- نظام فلترة ذكي للتصنيفات
- مؤشر تقدم القراءة في صفحة المقال
- زر "استمع للمقال" مع إمكانية الإيقاف والمتابعة ودعم المتصفحات القديمة
- أزرار مشاركة اجتماعية
- نظام تعليقات أنيق وسهل الاستخدام
- تنقل سلس بين الصفحات

### ⚡ الأداء والتحسين
- سرعة تحميل محسنة
- تحسين محركات البحث (SEO)
- دعم Schema Markup
- تحميل تدريجي للصور
- ضغط ملفات CSS و JavaScript

### 🔧 لوحة التحكم
- خيارات تخصيص شاملة
- تحكم في الألوان والخطوط
- إعدادات الأداء والمحتوى
- إظهار/إخفاء العناصر
- CSS و JavaScript مخصص

### 🎨 دعم Elementor
- توافق كامل مع Elementor المجاني
- ودجات مخصصة للمقالات
- دعم الهيدر والفوتر المخصص
- تخطيطات مرنة

## متطلبات النظام

- **ووردبريس:** 5.0 أو أحدث
- **PHP:** 7.4 أو أحدث
- **MySQL:** 5.6 أو أحدث
- **الذاكرة:** 128MB كحد أدنى (256MB مُوصى به)

- Elementor (مجاني)

## التثبيت

### الطريقة الأولى: رفع ملف ZIP

1. **تحميل القالب:**
   - احصل على ملف `muhtawaa-theme.zip`

2. **رفع القالب:**
   ```
   لوحة الإدارة → المظهر → القوالب → إضافة جديد → رفع قالب
   ```

3. **اختيار الملف:**
   - اختر ملف `muhtawaa-theme.zip`
   - انقر على "تثبيت الآن"

4. **تفعيل القالب:**
   - انقر على "تفعيل" بعد اكتمال التثبيت

### الطريقة الثانية: رفع عبر FTP

1. **استخراج الملفات:**
   - استخرج محتويات `muhtawaa-theme.zip`

2. **رفع المجلد:**
   ```
   /wp-content/themes/muhtawaa-theme/
   ```

3. **تفعيل القالب:**
   ```
   لوحة الإدارة → المظهر → القوالب → Muhtawaa → تفعيل
   ```

## الإعداد الأولي

### 1. إعداد القوائم

```
لوحة الإدارة → المظهر → القوائم
```

- أنشئ قائمة جديدة باسم "القائمة الرئيسية"
- أضف الصفحات المطلوبة
- اختر موقع "القائمة الرئيسية" في إعدادات القائمة

### 2. إعداد الودجات

```
لوحة الإدارة → المظهر → الودجات
```

- منطقة الشريط الجانبي: أضف ودجات المقالات الحديثة والتصنيفات
- منطقة الفوتر: أضف ودجات الروابط والمعلومات

### 3. إعداد التخصيص

```
لوحة الإدارة → المظهر → تخصيص
```

#### الألوان والتصميم
- **اللون الأساسي:** #4A90E2 (افتراضي)
- **لون النص:** #2C3E50 (افتراضي)
- **لون النص الثانوي:** #7F8C8D (افتراضي)
- **لون الروابط في الوضع الليلي:** #66B3FF (افتراضي)

#### الخطوط
- **نوع الخط:** Tajawal (افتراضي)
- **حجم الخط الأساسي:** 16px
- **حجم خط العناوين:** 32px
- **حجم خط الجوال:** 17px (افتراضي)

#### التخطيط
- **عدد أعمدة البطاقات:** 3 (افتراضي)
- **عدد المقالات في الصفحة:** 6
- **نصف قطر الأزرار:** 8px

#### العناصر المرئية
- ✅ إظهار شريط التصنيفات
- ✅ إظهار زر الوضع الليلي
- ✅ إظهار الفوتر
- ✅ إظهار وقت القراءة
- ✅ إظهار أزرار المشاركة

#### مؤشر القراءة والصوت
- **لون شريط التقدم:** قابل للتعديل
- **ارتفاع الشريط:** 4px (افتراضي)
- **نوع صوت القراءة:** أنثى عربية (افتراضي)
- **سرعة الصوت:** 1x (افتراضي)

### 4. خيارات القالب المتقدمة

```
لوحة الإدارة → المظهر → خيارات القالب
```

#### الإعدادات العامة
- ✅ تفعيل التحميل التدريجي للصور (مع آلية احتياطية)
- ✅ تفعيل التمرير السلس
- ✅ إظهار زر العودة للأعلى

#### إعدادات الأداء
- ✅ إزالة معاملات الاستعلام
- ⚠️ ضغط ملفات CSS (للمواقع المتقدمة)
- ⚠️ ضغط ملفات JavaScript (للمواقع المتقدمة)

#### إعدادات SEO
- ✅ تفعيل Schema Markup
- ✅ توليد Article Markup للمقالات
- ✅ تفعيل Open Graph
- **معرف Google Analytics:** أدخل معرف GA4

#### إعدادات المحتوى
- **طول المقتطف:** 25 كلمة
- **سرعة القراءة:** 200 كلمة/دقيقة
- **عدد المقالات المشابهة:** 3

## استخدام القالب

### إنشاء المقالات

1. **إضافة مقال جديد:**
   ```
   لوحة الإدارة → المقالات → إضافة جديد
   ```

2. **إعداد المقال:**
   - **العنوان:** اكتب عنوان المقال
   - **المحتوى:** اكتب محتوى المقال (يُفضل 200-300 كلمة)
   - **المقتطف:** اكتب ملخص قصير (25-30 كلمة)
   - **الصورة المميزة:** أضف صورة بحجم 800x600 بكسل
   - **التصنيفات:** اختر التصنيف المناسب
   - **الوسوم:** أضف وسوم ذات صلة

3. **إعدادات خاصة:**
   - **مقال مميز:** ضع علامة في خانة "مقال مميز" لإظهاره في القسم المميز

### استخدام Elementor

1. **تحرير الصفحة:**
   ```
   انقر على "تحرير باستخدام Elementor"
   ```

2. **الودجات المخصصة:**
   - **المقالات المميزة:** عرض المقالات المميزة
   - **المقالات الحديثة:** عرض أحدث المقالات
   - **تصنيفات المقالات:** عرض التصنيفات

3. **إعدادات الودجات:**
   - **عدد المقالات:** حدد عدد المقالات المعروضة
   - **عدد الأعمدة:** اختر تخطيط العرض
   - **إظهار المقتطف:** تحكم في عرض المقتطف
   - **إظهار وقت القراءة:** تحكم في عرض وقت القراءة

### تخصيص الألوان

1. **الألوان الأساسية:**
   ```css
   :root {
     --primary-color: #4A90E2;
     --text-color: #2C3E50;
     --text-secondary: #7F8C8D;
     --secondary-color: #F8F9FA;
   }
   ```

2. **ألوان الوضع الليلي:**
   ```css
   [data-theme="dark"] {
     --dark-bg: #1a1a1a;
     --dark-card-bg: #2d2d2d;
     --dark-text: #ffffff;
   }
   ```

### إضافة CSS مخصص

```
لوحة الإدارة → المظهر → خيارات القالب → CSS مخصص
```

مثال على تخصيص الألوان:
```css
/* تخصيص لون الروابط */
a {
    color: #e74c3c;
}

/* تخصيص خلفية البطاقات */
.article-card {
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
    color: white;
}

/* تخصيص حجم الخط */
.article-title {
    font-size: 1.5rem;
}
```
